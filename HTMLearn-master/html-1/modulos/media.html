HTML Video

O elemento de Html <video> é usado para mostrar um video numa página web

Exemplo:

<video> style="width:600px;max-width:100px;" controls>

  <source src="video.mp4" type="video/mp4">

  <source src="video.ogg" type="video/ogg">

  Your browser does not support HTML5 video.

</video>


O atributo controls adiciona os controlos de video, tais como reproduzir, pausa e volume.

O elemento <source> permite especificar os ficheiros de video alternativos, dos quais o browser poderá escolher. Este usará o primeiro formato que reconhecer.

Para começar um video automaticamente podemos utilizar o atributo autoplay.

<video> style="width:600px;max-width:100px;" autoplay>

  <source src="video.mp4" type="video/mp4">

  <source src="video.ogg" type="video/ogg">

  Your browser does not support HTML5 video.

</video>

Adiciona muted depois de autoplay para o video começar automaticamente mas sem som.

HTML Imagens

Imagens servem para melhorar o design e aparência da vossa página web.

A tag <img> é usada para colocar uma imagem na página web.

A tag <img> é vazia, apenas contém atributos e não tem uma tag para fechar.

A tag <img> tem dois atributos obrigatórios, "src" que especifica o caminho para a imagem, e "alt" que especifica um texto alternativo para a imagem

Exemplo:

<img src="url" alt="textoalternativo">

Podes ainda acrescentar os atributos "width" e "height", tal como os nomes indicam permitem-nos mudar o comprimento e a altura e são sempre definidos em pixeis
