Formulários

São usados para colecionar inputs do utilizador.

Estes permitem a introdução de valores de diferentes tipos.

São definidos através da tag <form>

Um input pode ser mostrado de diferentes maneiras através do atributo type

Alguns exemplos
Input Descrição
<input type="text"> Um input para texto
<input type="radio"> Mostra um radio button
<input type="checkbox"> Mostra uma checkbox
<input type="submit"> Mostra um butão de submit
<input type="button"> Mostra um butão clicável

Para além do elemento input o elemento form conta ainda com outros elemnetos

Tag Descrição
textarea Define um controlo de input de texto multilinha
label Define uma label para um controlo de input
fieldset Agrupa elementos relacionados num formulário
legend Define uma legenda para um controlo fieldsett
select Define uma lista pendente
