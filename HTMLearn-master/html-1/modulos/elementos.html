Elementos

Os elementos html é tudo aquilo que vai desde o início da tag até ao final.

Exemplos:

<h1>O meu primeiro cabeçalho</h1>
<p>O meu primeiro parágrafo</p>


Existem elementos que podem não ter conteúdo como por exemplo o <br>”.

As tag <h1> a <h6> definem cabeçalhos de diferentes tamanhos, sendo que o <h1> é o maior e o <h6> o menos.

A tag <h1> define um paragrafo.

A tag <div> é usada para dividir o documento em secções.

Alguns elementos vão aparecer corretamente mesmo que não seja colocada a tag para fechar, contudo não devemos confiar pois podem aparecer erros mais tarde se nos esquecermos de fechar as tags.
