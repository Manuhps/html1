O principal objetivo do nosso site é uma melhor aprendizagem do HTML. Temos vídeo-tutoriais de diferentes módulos, como por exemplo elementos, tabelas, media e formularios, para que o utilizador possa aprender um pouco de tudo. No HTMLearn também temos leaderboards para comparares os teus pontos com os dos teus amigos. Após completar todos os exercícios recebes as nossas badges especiais!
